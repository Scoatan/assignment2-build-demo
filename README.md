# Assignment 3 - Unit Testing Demo

This project demonstrates the implementation of a unit testing framework in a Java Maven workflow.

## Tools Used

- Ubuntu Linux
- Eclipse IDE
- Java 17
- Maven
- JUnit 5

## What Was Added

The original Maven project was updated to include JUnit 5 as a testing framework. A test file named `AppTest.java` was created under `src/test/java/com/johnathan/homework`.

## Unit Testing Explanation

Unit testing is the process of testing small parts of a program, such as methods or classes, to confirm that they behave correctly. In this project, JUnit tests verify that the application returns the expected message and correctly adds two numbers.

## How to Run the Tests

```bash
mvn test
```text
